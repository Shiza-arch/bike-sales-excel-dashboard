# bike-sales-excel-dashboard
🚴‍♂️ **My First Data Analytics Project — Bike Sales Dashboard**

I’m excited to share my beginner **Data Analytics project** as part of my learning journey.

For this project, I worked with a Bike Sales dataset from Kaggle and followed the complete analytics workflow:

✅ **Data Cleaning** – prepared and organized raw data for analysis
✅ **Pivot Tables** – summarized customer and sales insights
✅ **Data Visualization** – created interactive charts
✅ **Dashboard Design** – built an Excel dashboard using slicers for dynamic filtering

📊 **Key Dashboard Insights**

* Average Income of Customers who purchased bikes
* Customer Age Analysis
* Customer Commute Distance Patterns
* Interactive slicers for **Marital Status, Region, and Education**

This project helped me strengthen my skills in:

* Excel Data Analysis
* Pivot Tables & Charts
* Dashboard Creation
* Data Storytelling
